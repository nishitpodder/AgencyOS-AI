# Approval Workflow

---

# Purpose

Before any proposal or email is sent, the user must review and approve the AI-generated content.

---

# Workflow

AI Generates Proposal
↓

User Reviews Proposal
↓

User Can:

- Edit Proposal
- Edit Email
- Approve
- Reject

↓

If Approved

Proposal Sent to Client

↓

Activity Logged

---

# Screen Sections

## Proposal Preview

Displays:

- Client Name
- Proposal
- Budget
- Timeline
- Scope

---

## Email Preview

Displays:

- Subject
- Recipient
- Email Body

---

## Action Buttons

- Edit Proposal
- Edit Email
- Approve
- Reject

---

## Confirmation Popup

"Are you sure you want to send this proposal?"

Buttons:

- Yes
- No

---

## Success Message

Proposal Approved Successfully.

Email Sent Successfully.

---

## Error Messages

Approval Failed.

Email Sending Failed.
