# Layout Rules

## Desktop

- Sidebar visible
- Multi-column layout
- Full dashboard widgets
- Complete tables

---

## Tablet

- Collapsible sidebar
- Two-column layout
- Scrollable tables

---

## Mobile

- Hamburger menu
- Single-column layout
- Cards instead of wide tables
- Large touch targets

---

# Dashboard

Desktop

- Four statistics cards in one row

Tablet

- Two cards per row

Mobile

- One card per row

# Performance

- Lazy load heavy components
- Optimize images
- Minimize API requests
- Cache dashboard data
